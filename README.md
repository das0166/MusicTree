## 주요기능
- API를 사용하여 인기 순위 음악과 최신 음악 등을 확인할 수 있게 함
- 메뉴, 버튼 등에 호버했을 경우 위치를 나타낼 수 있게 색 차이를 두었으며 회원가입 시 필수항목을 미작성하였을 경우 사용자에게 알려줌
